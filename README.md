# Bullet Planner

## Projekti kirjeldus
Bullet Planner on veebirakendus, mis aitab hallata igapäevaseid ülesandeid ja sündmusi ühes kohas.  
Rakendus ühendab kalendri, märkmed, sündmused ja fotod, võimaldades kasutajal lihtsalt oma tegemisi organiseerida.  
Rakendus on loodud **Next.js** ja **Supabase** abil.

## Põhifunktsioonid
- **Kalender:** lisa, muuda ja kustuta sündmusi kiiresti  
- **Märkmed:** loo, muuda ja korralda märkmeid  
- **Sündmused:** määra aeg, koht ja meeldetuletused  
- **Fotod:** lisa pilte märkmetele ja sündmustele  
- **Kasutajakontod:** turvaline registreerimine, sisselogimine ja profiili haldamine
